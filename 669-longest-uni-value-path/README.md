669. Longest Uni-value path
Easy
Given a binary tree, find the longest path that all nodes have the same value, return the length of the path. The longest path may include or not include root node.

Example 1:

         1

       /   \

    2       3

 /    \        \

2     2         3

Output: 3

Example 2:

         3

       /   \

    2       3

 /    \        \

1     2         3

Output: 3
